## Usage

Place product images in this folder and reference them in the landing page using:
```tsx
<Image src="/products/[filename]" alt="Product description" />
```

## Image Requirements
- Format: JPG/PNG
- Recommended size: 1200x800px minimum
- Aspect ratio: 3:2 or 16:9
- File size: < 500KB (optimized)
